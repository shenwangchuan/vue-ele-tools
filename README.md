# vue-ele-tools
基于vue+elementUI的自定义组件

## 安装
npm install vue-ele-tools

## 使用
在main.js文件中  
import VElTools from 'vue-ele-tools/src/components/index'  
Vue.use(VElTools)

## 内容
 1.树形下拉选择器(v-el-tree-select)  
 2.数据表格（v-el-table）  
 3.数据表单（v-el-form）
